# Customer Churn EDA

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a telecom customer churn dataset.

## Dataset
Telco Customer Churn Dataset

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed
1. Churn Distribution
2. Contract Type vs Churn
3. Monthly Charges vs Churn
4. Internet Service vs Churn
5. Payment Method vs Churn
6. Tenure Months vs Churn
7. Senior Citizen vs Churn
8. Paperless Billing vs Churn

## Key Insights
- Month-to-month customers churn more.
- Fiber optic users show higher churn.
- Electronic check customers churn more.
- Customers with shorter tenure are more likely to churn.
